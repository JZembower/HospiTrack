# HospiTrack
A transparent, real-time, and interactive emergency care dashboard that connects patient to nearby walk-in health care services based on wait times and queue length.

Begin by installing the virtualenv packages on the target machine with: 
    pip install -r requirements.txt

Next, run the following command: 
    streamlit run app.py
